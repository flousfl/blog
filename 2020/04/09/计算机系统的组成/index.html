    计算机系统的组成

    简述计算机系统的组成

    一个完整的计算机系统:

    • 硬件系统：

      展开

      • 中央处理器（CPU）

        • 控制器
        • 运算器
        • 存储器

      • 输入设备

      • 输出设备

    • 软件系统：

      展开
      • 系统软件
        • 操作系统（OS）
        • 程序设计语言
        • 系统检测诊断程序
        • 编译系统
        • 通用数据库管理系统等
      • 应用软件
        • 管理软件
        • 办公软件
        • 应用数据库管理软件
        • 图形图像处理软件
        • 为用户需求开发设计的软件

    计算机工作原理

    • 工作原理：
      • 程序存储和程序控制
    • 主要内容
      • 计算机硬件设备由运算器、控制器、存储器、输入设备和输出设备五部分组成
      • 计算机内部采用二进制
      • 程序存储和程序控制
        读入数据并运行=>暂存结果=>输出结果

    冯·诺依曼计算机的结构(接受不了这个图的，去翻书本p7)

    graph LR
  输入数据或指令==>输入设备
  输入设备==>存储器
  控制器==>存储器
  控制器--控制信号-->输入设备
  控制器--控制信号-->存储器
  控制器--控制信号-->输出设备
  控制器--控制信号-->运算器
  存储器==>输出设备
  输出设备==>输出数据或指令
  存储器==>控制器
  运算器==>存储器
  存储器==>运算器

    微型计算机的工作过程

    graph LR
  CPU访问输入输出接口-->读入数据并运行
  读入数据并运行-->暂存结果
  暂存结果-->输出结果

    自问问答

    简述CPU的组成，及其组成部分的功能。

    答：CPU由控制器、运算器和寄存器组成。
    运算器是负责对数据进行算术运算和逻辑运算的部件。
    控制器是计算机的控制中心，用来实现计算机本身运算过程的自动化，其作用是控制整个计算机的工作。其工作是取指令、解释指令以及完成指令的执行。
    寄存器是CPU的一个重要组成部分，它是CPU内部的临时存储单元，它用于存储运算中的数据、中间结果、地址和状态信息等。

    运算器的组成及功能

    • 算术逻辑部件（ALU）：用于完成算术运算和逻辑运算，以及移位、求补等操作
    • 累加器：暂存被操作数和中间运算结果。
    • 通用寄存器：运算时用于暂存操作数和数据地址。
    • 状态寄存器：也称标志寄存器，存放运算中产生的状态信息。

    控制器的组成及各部件的功能

    • 指令指针寄存器：用于产生及存放下一条待取指令的地址。
    • 指令寄存器：用于存放正在执行的指令。
    • 控制逻辑电路：用于产生执行微操作所需要的控制信号。
    • 时钟控制电路：用于产生指令执行过程中所需要的时间控制信号。

    简述存储器的分类

    1. 按存储介质的材料分类:半导体存储器，磁表面存储器和光存储器。
    2. 按存储器的工作方式分类：随机读写存储器、顺序读写存储器和只读存储器
    3. 按计算机系统结构分：内存储器和外存储器

    简述内存储器和外存储器各自功能和特点

    答：内存储器的作用是在计算机工作中存储正在运行的程序和程序所需要的数据。内存储器是CPU可直接进行访问的存储器，其特点是采用大规模及超大规模集成电路制造工艺，具有体积小、重量轻和存取速度快等特点。
        外存储器的作用是长期保存暂时不用的程序和数据，CPU不用直接访问，其特点是存储容量大、成本低、存取速度慢、可以永久的脱机保存信息。

    显示器的主要指标是什么? P13

    • 像素
    • 分辨率
    • 点间距
    • 灰度级
    • 屏幕尺寸
    • 对比度
    • 帧频
    • 行频
    • 扫描方式

    内存储器

    简述内存储器的组成及各部分特点。P10~12

    答：内存储器由随机读写存储器（RAM）、高速缓冲存储器（Cache）和只读存储器（ROM）组成。

    RAM：是在计算机工作中存储正在运行的程序和程序所需要的数据；断电后数据丢失且不可恢复；
    Cache：是CPU与主存储器之间的一种规模较小但速度很快的存储器，它的内容是随机存储器（RAM）中部分存储单元内容的副本;断电后数据丢失且不可恢复。
    ROM：数据不易丢失，即使计算机断电后ROM存储单元的内容依然保存；存储计算机系统中固定的程序和数据；

    只读存储器的分类和英文标示？P12

    答：只读存储的分类如下：

    1. 普通ROM
    2. 可编程只读存储器PROM
    3. 可擦可编程只读存储器EPROM
    4. 电可擦可编程只读存储器EEPROM
    5. 闪速存储器（Flash Memory）：本质上属于EEPROM

    RAM的分类

    RAM可分为

    • 静态随机存储器(SRAM)
      • SRAM的读写速度很快、集成度低、生产成本高。
    • 动态随机存储器(DRAM)
      • DRAM需要不断刷新，常用的RAM一般是DRAM，，SRAM主要用作Cache上。
    • 目前DRAM的主要参数：
      • 存储容量
      • 工作频率

    高速缓存存储器（Cache）

    高速缓存存储器（Cache）是为了解决CPU和内存之间的速度匹配问题，是一种规模小但速度快的存储器，一般采用静态随机存储器SRAM，一般集成在CPU中。

    输入输出设备

    投影机

    • CRT(阴极射线管)
    • LCD（液晶显示）
    • DLP（数字光学处理）
    • DLV（数码光路真空管 / 数字光阀）

    灰度级

    指像素的亮暗程度，彩色显示器的灰度级指颜色的种类。

    总线的组成和类型

    总线的分类有哪些? P15

    1. 按功能分
    • 内部总线
    • 系统总线
      • 数据总线（双向）
      • 地址总线（单向）
      • 控制总线（单向）
    • 外部总线
    2. 按传送数据格式分
    • 串行总线
    • 并行总线
    1. 按时序控制方式分
    • 同步总线
    • 异步总线

    总线的

    常用的微机系统总线中，ISA总线逐步被PCI总线所代替。PCI-E总线将全面取代PCI总线和AGP总线。

    PCI的优点：

    数据传输速率高、点对点串行连接、支持热拔插和热交换。
