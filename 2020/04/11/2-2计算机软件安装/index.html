    2.2.1 Windows 7操作系统的安装

    核心版本号为Windows NT 6.1

    安装分类

    • 升级安装

      • 指在保持原有设置的情况下
      • 将原有的操作系统升级为Windows 7
      • 保持计算机中原有的设置
      • 安装速度很快

    • 全新安装

      • 就是安装一个全新的操作系统

      • 这个系统与原来的系统毫无关系

      • 可以在Windows、DOS 系统、或没有操作系统的裸机上运行

      • 操作步骤

        • 启动Windows 7 安装程序

          • 需要在CMOS中将第一引导顺序设置为CDROM
          • 保存设置后，再将Windows 7 安装光盘放入光驱，重新启动计算机，按任意键用光驱引导安装光盘进入操作系统

        • 保持默认设置

          • 安装语言选择中文（简体）
          • 时间和货币格式选择而中文（简体，中国）
          • 键盘和输入方法选择 中文（简体）-美式键盘

        • 版本选择

          • 选择你希望安装的版本

        • 同意许可条款

        • 进入分区界面

        • Windows安装与配置

    • 无人值守安装

      • 指在安装的时候相关的设置全部自动化
      • 除了安装分区的选择需要自己手动来执行，其他操作全部自动完成
      • 是一个名为AutoUnattend.xml的文件在起作用，即无人值守应答文件

    驱动程序的安装和备份

    当驱动程序安装完毕后，接下来就需要安装硬件设备的驱动程序

    Windows操作系统的驱动程序通常由一些.vxd（或.386）、.drv、.sys、*.dll 等类型的文件组成

    驱动程序（device driver）

    • 全称设备驱动程序

    • 是一种可以使计算机操作系统和BIOS不支持的硬件设备进行通信的特殊程序，相当于硬件的接口

    • 操作系统只用通过这个接口，才能识别和控制相应的硬件设备工作，发挥出硬件设备的功能

    • 并不是所有用硬件设备都需要手动安装驱动程序

    • 如硬盘，显示器，光驱，键盘，鼠标等BIOS和操作系统能识别的设备就不需要另外安装驱动程序

    • 操作系统和BIOS的版本越高，不用额外安装驱动程序的设备就会越多

    • 硬件制造厂商会根据操作系统的不同版本和编写不同版本的驱动程序，并不断对驱动程序进行升级，保证该硬件与其他设备以及操作系统之间的兼容性，提示硬件的性能

    • 驱动程序安装

      • 用驱动安装官配那安装主板，集成网卡和声卡的驱动程序
      • 通过设备管理器安装和更新驱动程序的方法

    • 驱动程序状态

      • 黄色问号表示该硬件未能被操作系统识别
      • 黄色感叹号指该硬件未安装驱动程序或驱动程序安装不正确
      • 红色叉号表示该硬件可能被手动禁用或已损坏，或与系统存在严重的冲突，总之目前他不能正常工作

    • 通过设备管理器来安装和更新驱动程序

      • 右击需要更新的设备名称，在弹出“设备属性”对话框中单击“更新驱动程序”进行安装和更新
      • 双击需要插卡或更新的设备名称，在弹出设备属性对话框中单击“更新驱动程序”根据硬件更新向导进行驱动程序的安装和更新

    2.2.3硬盘分区和格式化

    硬盘分区

    在安装操作系统之前，必须先在计算机硬盘上创建一个主磁盘分区，然后在改分区上格式化文件系统

    硬盘分区的概念

    • 硬盘分区是指对硬盘物理存储空间进行逻辑划分，将一个硬盘分成一个或多个大小不等的独立逻辑
    • 在DOS Windows系统中用盘符（即26个英文字母加上冒号）来标识，由于“A:”和“B：”这两个盘符用来标识软驱，因此硬盘的这些逻辑区域（即分区）就用“C：”到”Z：“来表示
    • 在linux或其他类Unix系统中，IDE硬盘的分区表示为hd.、SCSI和SATA硬盘分区，移动存储设备表示为sd，比如第一个IDE硬盘主分区（或扩展分区）被命名为hda1-hda4，如果没有则跳过

    硬盘分区有五种类型

    • 主分区

      • 主分区是直接从硬盘上划分出的一块区域，通常位于硬盘的最前面一块区域。主分区中不能再划分其他类型的分区，因此每个主分区相当于一个逻辑磁盘，会被分配一个盘符

    • 扩展分区

      • 划出主分区后，硬盘剩余的部分可以划分为扩展分区，但扩展分区不能直接使用，必须将它划分为逻辑分区才能使用
      • 因此，扩展分区是没有盘符的，而每个逻辑分区都具有一个盘符。
      • 扩展分区和逻辑分区之间包含关系，一个硬盘最多只等创建一个扩展分区，而扩展分区可以划分为若干个逻辑分区，这些逻辑分区容量大小的综合等于扩展分区的容量

    • 逻辑分区

    • 非Dos分区

      • 在硬盘中非DOS分区是一种特殊的分区形式，他是将硬盘中的一块区域单独划分出来供另一个操作系统使用，一般存在于同时安装有Windows和Linux双操作系统的计算机上。
      • 非DOS分区不能被DOS和Windows操作系统所使用，只是一块被划分出去的存储空间，该分区只能被非DOS分区操作系统管理和使用

    • 活动分区

      • 活动分区是被激活的主分区，活动分区包括了操作系统启动所必须的文件和数据，如主引导程序等，用于启动和引导操作系统，活动分区会被操作系统和主板认定为第一个逻辑磁盘，对于Windows系统会被识别为C盘，二linux根据具体的设置而定

    硬盘分区格式

    硬盘格式化是指对磁盘中硬盘分区进行初始化的一种操作

    格式化分类

    • 低级格式化

      • 又称物理格式化

    • 高级格式化

      • 又称逻辑格式化

        • 根据用户指定的分区格式，在磁盘的特定区域写入特定数据，以达到初始化磁盘格式的目的

        • 包括

          • 对主引导记录中分区表相应区域的重写

          • 根据用户选定的文件系统，在分区中划分出一片用于存放文件分配表、目录表等用于文件管理的磁盘空间

          • 用户使用该分区管理文件

          • 高级格式化还会检查硬盘是否有坏扇区，扫描坏扇区的工作占据了格式化磁盘分区的大部分时间

          • 在对磁盘格式化时，操作系统还会提供一个“快速格式化选项”。所谓快速格式化只是在分区文件分配表中做删除标记，而不扫描磁盘以检查是否有坏扇区

            • 只有在硬盘以前曾被格式化过，并且能确保硬盘没有损坏的情况下，才可以使用此选项

      • 硬盘格式

        • FAT16

          • 是Ms DOS 和Windows95操作系统中常见的硬盘分区格式。
          • 他采用16位的文件分配表，能支持最大为2GB的分区

        • FAT32

          • FAT32格式采用32位文件分配表，增强了磁盘的管理能力，突破了FAT16对每一个分区的容量只有2GB的限制。
          • 在Windows XP系统中，由于系统限制，单个分区最大容量为32GB，
          • 在不超过8GB的分区中，FAT32分区的格式的每个簇容量都固定为4KB
          • 与FAT16相比，可以大大减少磁盘的浪费，提高吸盘利用率
          • 目前Windows 98/98/2000/XP/Vista/7/8/10等操作系统都支持这一系统分区格式
          • DOS不支持这种分区格式

        • NTFS

          • NTFS格式的安全性和稳定性极其出色，在使用中不容产生文件碎片
          • 并能对用户的操作进行记录，通过对用户权限进行非常严格的限制
          • 使每个用户只能按照系统赋予的权限进行记录，通过对用户权限进行操作，充分保护了系统与数据安全
          • NTFS是一个可恢复的文件系统，在NTFS分区上用户很少需要运行磁盘修复程序。NTFS支持对分区、文件夹和文件的压缩。
          • NTFS管理磁盘空间更有效率，当分区大小在2GB以上时，簇的大小都为4KB，最大限度地避免了对磁盘空间的浪费

        • Linux

          • 两种分区

            • Linux native 主分区
            • Linux Swap 交换分区

          • 这两种分区格式和安全性与稳定性都非常好，死机的机会大大减少，但目前支持这一分区格式的操作系统只有Linux操作系统

    硬盘的分区和格式化操作

    目前常用的硬盘分区软件

    • DM
    • 魔术分区大师（PQ）
    • 诺顿分区大师（PM）

    硬盘分区软件功能

    • 在不损失硬盘中已有的数据的前提下实现对硬盘重新分区
    • 格式化分区
    • 复制分区
    • 移动分区
    • 转换分区

    2.2.4计算机应用软件安装

    应用软件的分类

    按软件的获得方式分

    • 商业软件

      • 是指被当做商品进行交易的软件
      • 一般通过特定的渠道发行
      • 用户必须支付一定的费用，得到软件的开发者授权后才能使用

    • 共享软件

      • 是指软件开发者或自由软件开发商提供的以“先使用后付费”的方式销售的具有版权的软件
      • 最明显的优点是具有免费的试用期
      • 用户可以从各种渠道免费得到他的拷贝，适用满意后再付费
      • 共享软件在使用上有一定的限制，或限制使用时间，或限制功能，或添加水印等

    • 免费软件

      • 一般是软件开发商或自由软件开发者为了某些目的（如扩大公司影响力）免费向用户发放的产品
