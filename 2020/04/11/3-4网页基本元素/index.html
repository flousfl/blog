3.4.1网页元素、超链接与表格

网页元素

网页文本

  • 网页文本是网页的基本组成元素之一，是用户浏览网页时直观获取信息的最直接的手段
  • 通俗易懂的文本内容和合适的编排方式，可以更好的突出网页主题

网页图像

  • 网页图像在传达信息的通俗，也美化了网页，使网页内容更加丰富多彩，展现形式更加多样。相比较文本内容，图像容量更大，影响打开网页的速度，因此网页实际时采用图像，采用压缩类图像文件。
  • 目前常用的有JPG\GIF\PNG等
  • 网页使用的图像可以借助外部的一些软件，（如Photoshop）编辑后再使用，一些简单的图像编辑，如图像大小，边距，对齐等方式则可以在图像的属性面板调整

网页其他元素

  • 除了文字和图像外，网页中还可以插入各种多媒体对象，如声音，动画，视频等，从而创建更加生动的网页

超链接

超链接是通过文本，图像或热点区域从一个网页指向一个目标的链接关系、这个目标可以是另一个网页，也可以是相同网页上的不同位置，还可以是一个图像，一个电子有邮件路径，一个文件甚至是一个应用程序

超链接按路径的不同，可以分为三类：内部链接，锚记链接和外部链接。

如果按照对象的不同，网页中的链接又可以分为

  • 文本链接

    • 指通过单击文本实现的链接

  • 图像链接

    • 指通过单击图像实现的链接

  • e-mail链接

    • 也叫电子邮件链接，用户在浏览网页时，单击此链接，会弹出邮件应用程序插件

  • 锚记链接

    • 也叫书签链接，是页面汇总某些特定的对象，一般用在单个篇幅较长的网页，便于用户查看，属于本页链接

  • 空链接

    • 指的是一个无指向的链接

  • 脚本链接

    • 是执行javascript代码或调用JavaScript函数实现的链接

  • 热点链接

    • 也叫图像映射，属于一种形式的图像链接，是一个能对链接指示做出反应的图形，单击该图形的已经定义区域，可以跳转到与该区域相链接的目标

表格

在网页设计中，表格主要应用于两个方面：一个方面，作为数据列表进行显示，另一方面，表格可以控制各元素的高度，宽度，彼此之间的比例大小等，从而实现网页布局的划分，通过表格可以准确的设定各种内容，以实现各种不同网页的布局效果

一张表格称为行，纵向称为列；行列交叉的部分就称为单元格；单元格中的内容和边框直接的距离称为边距，别名：填充，

单元格直接的距离称为间距，整张表哥哥的边缘叫做边框，表格构成如图3-25所示

3.4.2文本添加与修饰

在网页中添加文本，有两种方法：

键盘输入。直接在打开的文档窗口中，通过键盘键入文本内容

复制素材文件，打开素材文件，复制所选文本，在DW文档窗口中将光标定外在要添加文本的位置，选择“编辑”菜单中的“粘贴”即可

除添加文本外，还有选取文本，删除文本，查找和替换文本等

  • 选取文本：

    • CTRL+A
    • 鼠标左键并拖动鼠标

  • 删除文本

    • 选定文档，通过“编辑”菜单栏中低端“清除”命令，或者按Delete键删除文档，

  • 查找和替换文本

    • 单击“编辑”菜单中的“查找和替换”选项，在弹出的对话框中，分布输入要查找和替换的文本。如需查找某段文本的位置，则单击“查找下一个”或查找全部，按钮，如需替换，则单击“替换全部”按钮。

设置文本格式

选中首行的标题“东岳泰山“单击”属性“面板，左侧的CSS按钮，设置”字体“按钮

文本的属性面板有多个选项，作用如下：

  • 格式：在格式下拉列表汇总可以设置如下几种文本格式

    • 无：系统默认的文字格式

    • 段落：段落格式，将文字转化为段落来处理

    • 标题1-6：标准的标题格式，数字越大，标题级别越低，字体越小

    • 圆形格式化的：自定义格式，可以根据需求对文本进行预格式化

      • 字体：用来设置文本字体
      • 大小：用来设置字体大小，“无”是系统默认大小
      • B 加粗
      • I 倾斜
      • 文本对齐按钮，左对齐，居中对齐，右对齐，两段对齐
      • 颜色：用来设置文本颜色，单击□可以打开调色板，也可以通过在文本框中输入颜色十六进制代码来设置文本颜色
      • 项目列表，编号列表，文本突出，文本缩进
      • 链接：为文本添加超级链接，可以直接在文本框中输入链接目标，也可以使用十按钮指定一个文件作为链接，或者单击文件夹按钮，选择一个文件作为链接读写。
      • 目标：如果为文本创建了一个超链接，则可以在目标下拉列表中选择链接的打开方式

插入空格和水平线

插入空格

  • 将光标定位在段首，选择“插入”面板上的文本选项，再选择“字符”中的“不换行空格”；如在文本直接插一个空格时，只需要直接按Space键即可，如需插入多个空额，则可以使用快捷键Shift + Ctrl + Space 键

插入水平线

  • 网页中可以使用水平线将不同的内容隔开，使网页的内容更加层次分明。水平线一般需要设置宽度、高度等属性。
  • 将光标定位在正文的最后，按Enter键另起一行，宽度一般与标题宽度等同；在网页下方版权信息处设置水平线，可以分割网页内容与版权信息

3.4.3图像添加与修饰

插入图像

单击插入菜单中的图像“命令”或选择”插入面板上常用”选项中的“图像”按钮，在弹出的选择图像源文件对话框中，选择images文件夹中的ts.jpg文件，单击“确定”按钮，如图3-31所示

设置图像属性

图像属性面板中显示了图像许多常用属性，作用如下

  • ID

    • 输入图像名称，供编写脚本时引用

  • 宽、高

    • 分别表示宽度和高度，指定图像被载入页面时所需的空间大小，默认以像素为单位，如果设置的宽度和高度与图像实际大小不符，则图像在浏览器中按设置的大小显示
    • 虽然改变了图像的宽度于高度，但不会改变图像的容量大小，即不会改变图像下载时间

  • 指定图像的源文件，单击文件夹按钮，在弹出的对话框中选择文件，或者爱文本框中直接输入图像的路径及文件名

  • 链接

    • 为图像指定超链接，可以直接在文本框汇总输入链接目标，也可以使用十按钮指定一个文件作为链接，，或单击文件夹按钮，选择一个文件作为链接对象

  • 替换

    • 指定显示在图像位置上的可选文字，当浏览器无法显示图像时，显示这些文字

  • 编辑

    • 设置一个图像尾部的编辑器，可以用来修改图像

  • 地图制图指针热点工具

    • 用于设置图像的热点链接

选中Flash动画，在属性面板中，提供了Flash动画差用属性设置功能，主要属性如下所示

  • 名称：为插入Flash文件命名，供编写脚本时引用

  • 宽、高：

    • 用于定义Flash动画吓死尺寸，也可以通过在文档中拖动缩放手柄来改变其大小

  • 文件：

    • 用于之你打过Flash动画文件的路径

  • 循环：

    • 如果勾选该选项，动画将在浏览器端循环播放

  • 自动播放：

    • 如果勾选该选项，文档在浏览器载入时候，Flash动画将自动播放

  • 垂直边距、水平边距：

    • 用于定义Flash动画边框与周边内容之间的距离，以像素为单位。

  • 品质：

    • 用于设定Flash动画与周围内容的对齐方式

  • 背景颜色：

    • 用于设置当前Flash动画的背景颜色

  • 编辑：

    • 用于打开Flash软件对源文件进行处理，使用编辑时，要确保源文件*.falsh存在

  • 播放/停止：

    • 用于在设置视图中播放或停止Flash动画。

  • 参数：

    • 用于设置使Flash能够顺利运行的附加参数

  • 当网页中包含两个以上的Flash动画时，要预览所有Flash内容，可以按快捷键Ctrl + Alt + Shift +P

  • 选中文本，右击选择“创建链接”，在弹出的选择文件对话框中。选择目标文件，单击“确定按钮”
  • 选中文本，在“属性”对话框中，“链接”输入目标文件即可

创建好超链接后，可以通过修改属性面板上的目标来调整超链接打开方式

  • _blank和new

    • 把超链接指向的内容在同一个新窗口中显示，blan始终在不同窗口显示，new始终在同一窗口显示

  • _parent

    • 把链接指向的内容装入当前页父窗口中

  • _self

    • 将链接指向的内容装载到当前页的窗口或框架中

  • top

    • 完全取代当前页的所有框架

设置图像链接

除了文字有链接功能外，图像也能够设置超链接

创建图像超链接

  • 选中页面中的“东岳泰山”介绍照片ts.jpg右击弹出快捷菜单

创建图像热点超链接

  • 选中页面最左边的图片，在其属性面板汇总左下方的地图选项中选择“矩形热点超链接”

热点链接

  • DWcs6提供了矩形，圆，多边形等三种热点图形方式。对所键的热点链接不满意，也可以任意修改或删除

移动链接位置

  • 选中图片“属性”面板，“地图”中的箭头，指向已建热点区域并单击即可选中它，然后拖动鼠标可将热点图形移动至图片任意位置，从而重新调整图片中的热点部位

创建关闭窗口脚本链接

JavaScript是一种基于对象和事件驱动并具有安全性的脚本语言，通过嵌入或调入到标准 HTML语言中实现，通过JavaScript可以实现多种生动的链接，如弹出警告框脚本链接，收藏链接，打印链接和关闭窗口链接

创建关闭窗口链接：

  • 在水平线下风输入“关闭擦窗口文本”，选中该文本，在属性面板，“链接”文本框中输入代码“JavaScript：Windows.close()”
  • 仅IE有效

3.4.6创建表格

插入表格

单击插入中的“表格”命令，在弹出的表格对话框中设置

各项作用分别如下

  • 行数、列和表格宽度：

    • 分别用于设置表格的行数

  • 边框粗细：

    • 用于设置表格边框的宽度，默认以像素为单位。如果设置此选项为0，则浏览伟业时不显示表格边框。

  • 单元格边距：

    • 用于设置单元格与单元格之间的距离。

  • 单元格间距：

    • 用于设置相邻单元格之间的距离

  • 标题：

    • 设置表头的位置

  • 辅助功能：

    • 可以输入表格的标题以及选择标题的对齐方式

表格属性设置

方法1：

  • 将光标定位在表格中的一个单元格中，单击窗口底部标签选择器中与该表格对应的标签，即可选中该表格

    方法2：

    • 将光标移至表格边框，点指针变成双向箭头时，点击鼠标左键即可选中该表格

    方法3：

    • 将光标移动至表格外部靠近表格，当表格变为表格和箭头组合形状时，单击鼠标左键即可选中该表格

    选中该表格后，在表格属性面板设置各选项的作用如下

    • 表格

      • 用于设置表格的名称

    • 填充

      • 也称为点云哥边距，用于设置表格中单元格内容到边框的距离

    • 间距

      • 也称为单约个间距，用于设置相邻单元格之间的距离

    • 对齐

      • 用于设置表格的对齐方式，主要有左对齐，居中对齐和右对齐方式，默认为左对齐

    设置单元格属性

    单元格属性面板

    • 水平

      • 设置单元格中内容的水平对齐方式

    • 垂直

      • 设置单元格中内容的垂直对齐方式

    • 宽和高

      • 以像素为单位或以浏览器窗口的宽度的百分比来设置单元格的宽度和高度，

    • 不换行

      • 设置单元格文本是否换行。如果勾选“不换行”复选框，当输入的数据超出单元格宽度时，会自动增加单元格的宽度来容纳数据

    • 标题

      • 勾选复选框，将行或列的每个单元格的格式设置为标题单元格格式。

    • 背景颜色

      • 用于设置单元格的背景颜色

    单元格选择操作

    选择连续单元格：可以通过鼠标拖动的方式选择连续的单元格；也可以通过将光标定位到某一单元格，按住Shift键，单击另外一个单元格的方式，选择两个单元格之间连续的区域。

    选择不连续的单元格：按住Ctrl键，单击另外一个单元格的方式，选择两个不连续的单元格区域

    选中单元格后右击，在弹出的快捷菜单的“表格”选项中可以设置合并、拆分单元格，添加、删除行和列

        1. 1.3.1. 在网页设计中，表格主要应用于两个方面：一个方面，作为数据列表进行显示，另一方面，表格可以控制各元素的高度，宽度，彼此之间的比例大小等，从而实现网页布局的划分，通过表格可以准确的设定各种内容，以实现各种不同网页的布局效果
        2. 1.3.2. 一张表格称为行，纵向称为列；行列交叉的部分就称为单元格；单元格中的内容和边框直接的距离称为边距，别名：填充，
