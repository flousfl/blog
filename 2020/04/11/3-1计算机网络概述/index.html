3.1.1计算机网络概念与功能

计算机网络

所谓计算机网络，是指将地理位置不同，且具有独立功能的多台计算机及其外部设备，通过信息设备和通信线路连接起来，在网络操作系统，网络管理软件，以及网络通信协议等

计算机网络存在的主要目的是在于解决计算机之间的数据通信和资源共享

计算机网络是一个复杂的软硬件系统，不仅包括计算机、通信设备、通信线路等相关硬件，还需要多种网络软件的密切配合和支持

计算机网络主要包括包含网络连接对象（即元件、网络传输介质、网络连接的控制机制）和网络的方式和结构四个部分

计算机网络的主要功能和作用

数据通信

  • 数据通信是

资源共享

  • 数据共享
  • 软件共享
  • 硬件共享

提高系统可靠性

  • 提高计算机系统可靠性主要体现在计算机网络中每台计算机都可以通过计算机网络相互称为后备节点，在某台计算机出现故障后，其他计算机可以立即运行由故障机承担的任务，从而避免系统的瘫痪

分布处理

  • 分布处理是指计算机网络中，用户可以合理选择计算机网络内的资源进行相应的数据处理，对于较为复杂的问题，还可以通过算法将任务分给不同的计算机处理，从而达到均衡网络资源的目的。此外，还可以通过算法将任务分给不同计算机处理，从而达到金很网络资源的目的，以并行的方式实现提高性能的运算，也被称为协同式计算机网络计算

负载平衡

  • 负载平衡。负载平衡是指在具有分布处理能力的计算机网络中，运行任务被相对均匀的分配给网络上的各台计算机，系统能根据计算机运行负载大小来转移任务，从而实现计算机网络负载的平衡

3.1.2计算机网络分类

按网络分布的地理范围

局域网

  • 也称局部网，是一种在小范围内实现的计算机网络，其网络分布范围一般在几百米到几十千米
  • 局域网结构相对简单，接入点通常在几十到几百个之间

城域网

  • 也称都市网，通常是在一个城市或地区内部实现的计算机网络，其网络分布范围一般在几十千米到上千米。
  • 城域网是局域网在地理范围上的延伸，其结构复杂，接入的计算机网络节点数目众多

广域网

  • 又称远程网，是一种覆盖范围比城域网更广的网络，可分布在一个省份内、一个国家，连接范围从几百千米到几千千米，可以分布在一个省份内，一个国家，典型的如全球最大的计算机网络，Internet

按网络速率与网络带宽

按网络传输速率可以分为高速网和低速网

在每秒几兆比特至每秒几比特范围的网络称为高速网络。

也可将传输速率在每秒几千比特的网络称为低速网

3.1.3计算机网络组成

网络软件系统

网络软件系统包括网络操作系统，通信软件，网络协议，和网络应用软件

网络操作系统运行与服务器上，除了具有一般操作系统的处理器管理，存储器管理，设备管理和文件管理外，还需具备以下两大功能，一是提供高效、可靠的网络通信功能，以实现计算机之间高效可靠的通信；二是为网上用户提供多种网络服务功能，如文件传输服务功能、电子邮件服务功能、远程打印服务功能等

目前常用的网络操作系统

  • Windows 系列

    • 由微软公司研发的网络操作系统主要有 Windows NT Server Windows 2000 Server /advance Windows 2003 Server 等

  • Unix 系列

    • 是一个强大的支持多用户，多任务，的网络操作系统。他支持多种处理器架构，属于分时操作系统，通常应用于大型企业，单位的局域网中

  • Linux 系列

    • 是一种类似UNIXb并基于源代码开放的网络操作系统。改系统已成为世界上使用最多的一种UNIX类操作系统，Linux是一款免费的操作系统，用户可以通过网络或其他途径免费获得，并可以任意修改其源代码。目前使用较广的Linux发型版本有RED hat fodora bebian ubyntu 红旗linux 等

通信软件

通信软件是用以监督和控制通信工作的软件，它不仅是计算机网络软件遵守的规则集合，简而言之，是为进行网络数据交换而建立的规则，标准或约定，是网络软件系统的重要组成部分

网络协议

一条网络协议至少包括三个基本要素

  • 语法

    • 约定数据与控制信息的结构和格式

  • 语义

    • 用以说明通信双方如何操作，及应答方式

  • 时序

    • 用以详细说明事件实现的先后顺序

目前常用的协议有：基于互联网的TCP/IP协议和局域网IEEE 802.X协议

TCP/IP协议现已成为网络互连的基本标准，包含两个核心协议：传输控制协议（TCP）和网际协议（IP）

IEEE 802.X协议是以太局域网接入标准协议，其中，IEEE 802.3协议是总线型局域网的标准

开放系统互联模型

计算机网路技术是伴随着各种计算机网络应用的需求而逐渐发展起来的，不同的网络系统有着各自不同的通信方式和内部网络编码。为了使计算机能够在异种网络间实现自由通信和资源共享，国际标准化组织简称ISO 在1981年提出了开放系统互联模型 OSI 这一标准的问世消除了不同网络系统间因协议不同而导致的通信障碍，使得在互联网范围内，不同的网络系统可以不需要转码的转换装置即可实现通信。这其中的“开放”是指任何两个准守参考模型和相关标准的系统能进行连接，而互联则指能够将不同的系统互相连接而实现共享资源，交换信息、分布处理和分布应用等网络应用

OSI是一个机遇网络协议规范化的网络工作逻辑参考模型

七层模型

  • 应用层

    • 主要功能是保持应用程序之间建立连接所需的记录，包含大量用于现实文件传输，电子邮件收发，远程登录以及其他各种软件应用的协议

    • 简而言之，应用陈层主要是实现以下终端服务的应用，如文件上传下载，网页浏览，及时通信等

    • 常见协议

      • FTP
      • DNS
      • SMTP
      • HTTP
      • NFS
      • Telnet

  • 表示层

    • 主要功能是为了让采用不同表示方法的计算机之间能够通信
    • 简而言之，表示层主要是对接收的数据进行解释或恢复，如数据编码，数据的加密与解密，数据压缩与恢复等

  • 会话层

    • 主要负责两个会话进程之间的通信。会话层让不同计算机上用户建立起会话关系，并对会话进行管理。会话层允许信息同时双向传输，或任意时刻只能单向传输。
    • 简而言之，会话层主要在系统之间发起会话或接受会话请求，这一会话可以基于IP地址，也可以是MAC或是主机名

  • 传输层

    • 从会话层接收数据，并在必要时把它分解为较小的单元，传递给网络层

    • 简而言之，传输层定义了数据传输的协议和端口号，如TCP协议，UDP协议，万维网WWW端口80等，并将从下层接收的数据进行分段传输，在达到目的地址后进行重组。在这一层，数据可视为段

    • 常见协议

      • TCP协议
      • UDP协议

  • 网络层

    • 主要功能是实现路由的选择，阻塞控制，以及异种网络的互联，

    • 简而言之，网络层主要是将接收到的数据进行IP地址的封装与解封装。在这一层数据可视为数据包，路由器，二层交换机、三层交换机工作于此

    • 常见协议

      • IP协议
      • ICMP协议
      • ARP协议
      • RARP协议

  • 数据链路层

    • 是OSI的第二层，主要功能是在两台主机间建立数据连接，向物理层传输数据信号，并使之对网络层呈现为一条无差错传输线路。
    • 在数据链路层还需实现流量控制，解决在双向传输数据时的收发双方数据帧竞争线路使用权问题以及解决广播式网络的共享信道访问的问题
    • 简而言之，就是将接收的数据进行MAC地址（网卡地址）的封装或解封装，在数据链路层，数据可视为帧，网桥，二层交换机工作与此

  • 物理层

    • 是OSI的第一层，也是最底层，主要功能是定义了网络的物理结构、传输的电磁标准、比特流的编码及网络的时间原则，如网线的接口类型、光纤的接口类型、各种传输介质的传输速率等，以实现比特流的传输（通过电流变换来传输数据，即模数转换与数模转换）
    • 简而言之，就是针对信号的实际物理传输设计相应的接口标准。在物理层，数据以比特流的形式存在

网络硬件系统

网络硬件系统包括

  • 网络服务器

    • 是向网络内工作站提供各类服务的计算机，是网络的核心。其最主要的作用是运行网络操作系统，同时存储，管理，网络郑的共享资源，并具有对各工作站的程序服务和监控功能，网络服务器具有速度快，容量大，可靠性高的特点

  • 网络工作站

    • 是用户和网络的接口设备
    • 可称为客户机
    • 通过网卡，通信介质以及通信设备连接到网络服务器，实现网络的访问和资源的获取

  • 网络传输介质

    • 是指在网络中传输信息的载体，作用是形成通信网络中发送方和接收方之间的一条物理通信线路

    • 有线

      • 双绞线

        • 是由两根相互绝缘的导线按照一定规格螺旋状缠绕而排列在一起的一种通用配线。

        • 双绞线可分为屏蔽双绞线（STP）和非屏蔽双绞线（UTP）

          • 非屏蔽双绞线抗电磁干扰能力弱，但具有节省空间，成本低，重量轻，易弯曲，易安装可以同时传输模拟信号和数字信号的有点，其传输距离一般为100米

        • 按粗细与直径规格可分为一类线至七类线，其中五类线最高传输率可达到100Mbps至1000Mbps,广泛应用于百兆和千兆以太网连接

      • 同轴电缆

        • 由两根轴心，相互绝缘的圆柱形金属导体构成的基本单元（同轴对），再由单个或多个同轴对组成的电缆

        • 从里到外一共有四层

          • 中心铜线
          • 塑料绝缘体
          • 网状导电层
          • 外层绝缘材料

        • 常用的同轴电缆有两类

          • 基带同轴电缆

            • 一般为细缆，阻抗为50欧
            • 用于数字传输，传输带宽为1~20MHz
            • 最大传输距离为180M
            • 要与50欧的T形连接器配合使用

          • 宽带同轴电缆

            • 一般为粗缆
            • 其阻抗为75欧
            • 适用于模拟传输，常用语CATV网（有线电视网）
            • 最大传输距离可达2000米以上

      • 光缆

        • 光缆是由光导纤维（光纤）缆芯、包层、吸收外壳和保护层四部分组成，外层是一层折射率较低的材料，按光在纤维中的传输模式，光纤可分单模光纤和多模光纤，单模光纤只能传送一种模式的光，有效传输距离可达数十千米
        • 多模光纤容许不同模式的光在一根光纤上传输，在千兆网中，多模光纤最高可支持550米的传输距离，在百以太网中，多模光纤最高可支持2000米的传输距离

    • 无线

      • 红外线
      • 微波
      • 卫星通信
      • 激光信道

  • 网络设备

    • 网络适配器

      • 也称网络接口卡或网卡

      • 是连接计算机与网络的硬件接口设备，网卡主要功能在于实现数据的封装与解封装，即将来自于本机的数据封装为数据包并向网络转发，

      • 每一块网卡在出厂时都会为其分配唯一一个物理地址

      • 按网卡支持的总线接口

        • ISA网卡
        • EISA网卡
        • PIC网卡
        • USB网卡

      • 按网卡的传输介质

        • RJ45网卡
        • BNC网卡
        • AUI网卡

      • 按网卡的速度

        • 10Mbps
        • 100Mbps
        • 10/100Mbps
        • 1000Mbps
        • 10000Mbps

    • 中继器

      • 也称集中器，工作与物理层，是两个网络节点之间进行物理信号双向转发的网络互连设备，他的主要作用是通过对传输信号的整形与放大，来避免干线上传输信号因衰减而失真，从而延长信号灯传输距离，扩展局域网的连接范围

    • 集线器

      • 又称网络转发器，是一种作为网络中暑连接各类节点，从而形成星形结构的一种网络设备，集线器的主要功能是对来自网络的信号进行再生，整形和放大，以扩大网络的传输距离，并把所有的节点连接起来，集线器的工作原理类似于中继器，工作在物理层

    • 交换机

      • 可以为接入交换机的任意两个网络节点提供独享的店信号线路，

    • 路由器

      • 工作与网络层，要求网络层以上的高层协议相同或兼容，用以实现不同类型的局域网互联，或实现局域网与广域网的互联。路由器可以实现网络层以下各各层协议的转换，除了具备网桥全部功能外，还有路由选择功能

    • 网关

      • 网关，又称为网间连接器、网间协议转换器，工作与OSI的多个层面，（传输层，会话层，表示层，应用层）
      • 用于提供协议转换，路由选择，数据交换等网络兼容共的网络互连设备
      • 用于提供协议转换，路由选择，数据交换等网络网络兼容的网络互连设备。网关基层了路由器的全部功能，同时还提供协议之间的转换，既可以用于广域网互连，也可以用于局域网互联

    • 网桥

      • 工作在数据链路层，实现多个网络协议相同或兼容的网络协议，其主要作用是将网络内多个网段加以连接，并对网络数据的流通进行过滤，转发隔离等操作者，从而调高网络的性能、可靠性和安全性

3.1.4计算机网络拓扑结构

计算机网络拓扑结构

网络拓扑结构是指在计算机网络中，各通信节点通过制定设备和线路进行相互连接的方法和形式，其核心内容是网络节点之间的连接关系。

目前常见的局域网拓扑结构主要有星型，总线型，环形，和网状型。

广域网采用网状拓扑结构

常见拓扑结构

星形

  • 由中央节点和通过点到点的链路连接到中央节点的各节点组成。他是以中央节点为中心，把若干外围节点连接起来的辐射式互联结构，很使用于局域网连接

  • 优点

    • 便于节点、设备的添加或删减
    • 由于每个接入点只连接一个设备，当某个节点出现故障时，不影响整个网络的运行

  • 缺点

    • 以中央节点为中心，将导致中央节点发生故障时整个网络不能运行

总线型

  • 以一根公用通信线路为传输介质，所有网络节点均通过相应接口直接连接到腾通信线路，这根所有节点通信公用的传输线路被称为总线。在局域网中，总线一般采用同轴电缆或双绞线

  • 优点

    • 通信线路的长度较短，易于安装和部线
    • 由于各个节点共用一条总线作为数据通路，信道利用率高
    • 便于扩充或删除节点

  • 缺点

    • 网络故障较难诊断
    • 总线长度和接入点数均有一定限制

环型

  • 优点

    • 每个节点的通信地位均平等
    • 容易实现远距离的高速数据传送
    • 电缆故障容易查找和排除

  • 缺点

    • 通信线路闭合的环结构使节点不易扩充
    • 可靠性差，环路断开将导致整个网络不能工作

网状型

  • 优点

    • 节点间的多链路连接增强了网络连接的可靠性
    • 部分节点或单条线路故障不影响全网运行

  • 缺点

    • 复杂而庞大的网状结构需要较高的建设和运行费用
    • 网状型拓扑结构的路径控制复杂，增加了算法设计难度
