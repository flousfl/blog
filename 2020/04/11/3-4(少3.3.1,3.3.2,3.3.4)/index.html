3.3.3基本标记应用

标题标记

格式

  • <h#>····</h#>

    • 其中取值范围1-6,1最大，6最小

作用

  • 设置文档各级标题

常用属性

  • align

    • 用于定义段落的对齐方式，默认左对齐。

    • left

      • 左对齐

    • righr

      • 右对齐

    • center

      • 居中对齐

分段标记

格式

作用

  • 使标记后面内容另起一段

常用属性

  • 用于定义段落的对齐方式，默认左对齐。

  • left

    • 左对齐

  • righr

    • 右对齐

  • center

    • 居中对齐

换行标记

常用属性

作用

  • 使文字后面内容换行显示

格式


水平线标记

格式

作用

  • 在文档中插入水平线

常用属性

  • align

    • 对齐方式，默认居中

  • color

    • 定义水平线颜色

  • noshade

    • 定义水平线为无阴影

  • size

    • 定义水平线宽度（粗细），单位默认为像素

  • width

    • 定义水平线的长度，单位为像素或百分比

字体标记

格式

作用

  • 设置标记间文本字体，大小，颜色等

常用属性

  • face

    • 用于设置标记间文本的字体

  • size

    • 用于设置标记间文本的字体，大小，颜色等

  • color

    • 用于设置标记间文本的颜色

  • title

    • 用于设置鼠标指向文本时出现的提示信息

图像标记

格式

作用

  • 在页面汇总插入图像文件，图像格式可以是GIF,JPG和PNG等

常用属性

  • src

    • 指定插入图像的名称或路径

  • alt

    • 为替换文本，当浏览器不能显示图像时，或在鼠标指向图像时，或图像加载时间过长时，所要替换的文本

  • width

    • 用于执行图像的宽度，单位是像素

  • height

    • 用于指定图像的高度，单位是像素

  • border

    • 用于指定图像的边框

  • hspace

    • 用于指定图像左右两侧与其他对象（如文本）之间的距离，单位为像素

  • vspace

    • 用于指定图像上下两侧与其他对象（如文本）之间的距离，单位为像素

  • align

    • 用于指定图像与文本的对齐方式，取值有top,middel,bottom,left,right,可以执行图像文字左右绕排的效果

超链接标记

格式

作用

  • 吧当前位置的文本，图像或热点区域链接到其他目标，这个目标可以是另一个网页，也可以是相同网页上的不同位置，还可以是一个图像、一个电子邮件地址，一个文件，甚至是一个程序

常用属性

  • href

    • 用于指定链接的目标地址，可以是站内文件，可以是某个网址等

  • target

    • 用于指定链接的目标窗口，其值可以是”blank”,”parent”,”self”,”top”

  • title

    • 为鼠标执行超链接时所显示的信息

表格的几种标记

格式

作用

  • 表格不仅用于数据的显示，而且是网页布局的常用工具

常用属性

  • table的常用属性

    • border

      • 指定表格边框的宽度（以像素为单位），如不设此属性，则边框的宽度默认为1像素

    • bordercolor

      • 指定表表格边框颜色

    • background

      • 设置表格的背景图像

    • bgcolor

      • 设置表格的背景亚瑟

    • width

      • 设置表格的宽度，其值可用像素表示，也可用占窗口的百分比表示

    • height

      • 设置表格的高度，其值可用像素表示，也可用占窗口的百分比表示

    • align

      • 设置表格在页面的水平位置，取值为left right center

    • cellspacing

      • 设置单元格直接的距离，其值可用像素来表示

    • cellpadding

      • 设置单元格内容与单元格边界直接的距离，其值可用像素表示

  • tr的常用属性

    • align

      • 设置表格在页面的水平位置，取值为left right center

    • valign

      • 设置行内容的垂直对齐方式，取值为top,middle,bottom，默认为居中对齐

    • height

      • 设置行高，其值可以用像素或表格高度百分比表示

    • title

      • 设置当鼠标指向该行时，显示的提示信息

    • bgcolor

      • 设置行的背景颜色

    • bordercolor

      • 设置行的边框颜色

    • nowrap

      • 设置本行所有内容不能自动换行

  • td的常用属性

    • align

      • 设置单元格内容的水平对齐方式

    • valign

      • 设置单元格内容的垂直对齐方式

    • heigh

      • 设置单元格的宽度可用像素或表格的百分比来显示

    • width

      • 设置单元格的垂直对齐方式

    • bgcolor

      • 设置单元格的背景颜色

    • bordercolor

      • 设置单元格的边框颜色

    • nowarp

      • 设置本单元格内容不能自动换行

    • rowspan

      • 设置单元格所占的行数

    • colspan

      • 设置单元格所占列数
