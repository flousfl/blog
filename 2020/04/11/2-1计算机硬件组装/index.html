2.1.1微型计算机的硬件组成

微型计算机概念

从外形上来看

  • 主机

    • 主机的概念

      • 一般将主机箱及颞部的部件统称为主机

    • 主机的结构

      • 主板

      • 内存条

      • 硬盘

      • 光驱

      • 显卡

      • 声卡

      • 网卡

  • 输入设备

    • 键盘

      • 键盘是计算机系统的基本输入输出设备

      • 键盘的分类

        • 按键数来分

          • 101键盘
          • 104键盘
          • 107键盘

      • 键盘的5个组成部分

        • 标准打字键区

        • 功能键区

        • 小键盘区

          • 具备两个功能，由NumLock决定

            • 灯亮为数字输入功能
            • 灯灭为上下左右、翻页功能

        • 光标控制区

        • 信号灯状态显示区

    • 鼠标

      • 按工作原理来分

        • 机械鼠标（已被淘汰）

  • 结构

    • 鼠标里有一个可以滚动的小球

  • 原理

    • 小球滚动摩擦两个滚轮，将滚动的距离转换为电信号

  • 缺点

    • 定位精度低
      - 容易磨损
- 寿命较短

  • 光电鼠标

    • 普通光电鼠标

      • 结构

        • 鼠标下方的小光源
          - 反射板
- 接收器

      • 原理

        • 鼠标移动时，光源发出的光线经过反射板后由接收器接收

      • 优点

        • 定位精度高
          - 防尘性能好
- 第一代要在特定是反射板上使用，第二代可以在除镜面以外的任何场所使用

    • 激光鼠标

      • 原理

        • 和普通光电鼠标一样，但用激光代替LED灯

      • 优点

        • 可以在任何表面使用

  • 扫描仪

    • 原理

      • 利用光电技术和数字处理技术，以扫描的方式将图形或图像信息转换为数字信号输入设备

    • 结构

      • 光源
        • 透镜
        • 电荷耦合器（CCD）
        • A/D转换器

    • 参数

      • 工作过程：
      1. 光源发出光线扫描原稿后被反射到一个透镜上
        1. 经过透镜打到电荷耦合器（CCD）上
        2. CCD产生输出电压
        3. 经过A/D转换成数字信号再送到计算机

  • 输出设备

    • 显示系统组成

      • 显示器

        • 显示器分类

          • 阴极射线管（CRT）显示器

            • 结构

              • 电子枪
              • 偏转线圈
              • 荫罩
              • 高压石墨电极
              • 玻璃外壳
              • 玻璃外壳上的荧光粉图层

  • 优点

    • 可视角度大

              - 无坏点
        - 色彩还原度高
        - 色度均匀
        - 可调节的多分辨率模式
        - 响应时间短
    - 缺点

- 液晶（LCD）显示器

    - 原理

        - 以电流刺激液晶分子产生点、线、面配合背部灯管构成画面

    - 结构

        - 彩色或黑白像素

      • 光源

            - 反射面

- 优点

    - 占用空间小

      • 能耗低

                - 辐射小
        - 画面柔

    - 缺点

- 等离子体显示器（PDP）

    - 原理

        - 采用了进几年来高速发展的等离子平面技术的新一代显示设备

    - 优点

        - 高亮度

      • 高对比度

            - 色彩还原性好
    - 厚度薄
    - 视角宽
    - 画面响应速度快

- 缺点

      • 显示卡

  • 相应软件

    • 打印机

      • 打印机的分类

        • 击打式打印机

          • 针式打印机

            • 原理

              • 利用打印钢针在纸上打印出由点阵组成的字符图形

            • 结构

              • 打印头

                • 9针

              • 16针

                • 24针

              • 字车

            • 输纸机构

              • 色带
              • 控制电路

            • 耗材

              • 色带

            • 优点

              • 耗材成本低

            • 能平面打印也可以打印多联票据

            • 缺点

              • 打印质量低

            • 噪音大

            • 应用领域

              • 银行

            • 超市

              • 用于票单打印的地方

        • 非机打式打印机

          • 喷墨打印机

            • 根据墨水的驱动方式不同

              • 分压电式

            • 热气泡式

          • 激光打印机

            • 原理

              • 利用激光扫描把要打印的字符在硒鼓上形成静电潜象，然后转成磁信号，使碳粉吸附在纸上，经定影后输出

            • 优点

              • 高质量的文字及图形

            • 打印速度快

            • 耗材

              • 碳粉

            • 硒鼓

          • 热升华打印机

            • 原理

              • 利用热能将四种颜色（CMYK）的固体颜料转印至打印介质上的仪器，通过半导体加热元件调节出不同的温度来控制色彩的比例和浓暗程度

            • 优点

              • 输出效果好

            • 缺点

              • 对打印的介质和使用环境要求很高

            • 打印成本高

              • 不适合打印文字、表格等

            • 耗材

              • 热升华墨水

            • 热转印纸

            • 应用领域

              • 打印彩色数码照片

      • 打印机的性能指标

        • 打印分辨率

          • 概念

            • 指打印机在输出时，横向和纵向上每英寸最多能打印的点数

          • 单位

            • DPI（每英寸打印点数）

          • 针式打印机

            - 180DPI

            • 360DPI

            • 喷墨打印机

              • 720DPI

            • 1440DPI

              • 2800DPI
              • 4800DPI

            • 激光打印机

              • 300DPI

            • 600DPI

              • 1200DPI
              • 2400DPI

        • 打印速度

          • 概念

            • 指打印机每分钟可以输出多少内容

          • 打印速度不是固定不变的，会受到不同打印对象，数据传输方式、打印机内存大小、计算机性能的影响

          • 单位

            • 针式

              • CPI（每英寸打印字符个数）

            • 喷墨

          • 激光

        • 打印幅面

          • 概念

            • 指打印机最大能够支持打印纸张的规格

          • 单位

            • A3

          • A4

            • 等

2.1.2计算机主机外部接口

电源接口

用途

  • 用于连接主机电源线
  • 是主机的供电接口
  • 计算机通过该接口与220V市电连接

形状

  • D形接口
  • 防止电源线插反

注意事项

  • 拔除电源时要先切断电源
  • 一手扶住机箱，一手抓住电源线轻拔
  • 拔除时可以摇晃

鼠标和键盘接口

PS/2接口

  • 是键盘和鼠标专用接口
  • 圆形6孔
  • 绿色为鼠标接口
  • 紫色为键盘接口
  • 不支持热插拔
  • 不能在带电情况下拔插鼠标，否则可能损坏接口电路

USB接口

显示器接口

VGA

  • 又称D-SUB
  • 15针的D形接口

DVI

  • 是数字视频接口
  • 无需将显卡产生的数字信号转换为模拟信号可以直接显示在屏幕上
  • 对于LCD显示器而言，DVI接口更有优势，将逐渐取代D-SUB端口

S-Video

  • 又称S端子
  • 全称Separate Video
  • 将亮度信号与色度信号分开传输，避免了亮度和色度互相干扰
  • 只能传输视频

USB接口

通用串行总线（英语：Universal Serial Bus，缩写：USB）

是1994年由因特尔、康柏、IBM、微软等多家公司联合提出的，目前已成功取代串口和并口

1994年发布USB0.7，至今已经USB3.2

USB1.1 -> 12Mbps

USB2.0 -> 480Mbps

USB3.0-> 5Gbps

优点

  • 使用方便

  • 支持热插拔

    • 热插拔指用户在不关闭系统，不切断电源的情况下拔出连接在主机的外部设备，或取出、更换损坏硬盘
    • 热插拔技术提高了系统对灾难的及时恢复能力、扩展性和灵活性等
    • 计算机的USB和IEE1394（火线接口）都支持热插拔

  • 连接灵活

  • 独立供电

  • 可连接最多127个设备

网线接口

超五类双绞线

  • 与计算机连接的接口是RJ-45，俗称水晶头
  • 是目前计算机连接网络最常用的接口

声卡接口

复制音频的输入、输出

大部分都使用小三芯接口

  • 红（音频输入）
  • 绿（音频输出）
  • 蓝（数字音频输入）

并行和串行接口

并行（LPT）

  • 简称并口
  • 25针D形接口
  • 目前针式打印机都用并行接口
  • 数据传输慢
  • 传输距离短（最大2米）
  • 不支持热插拔

串行（COM）

注意：专指COM口，非USB口

  • 与并口相比

    • 使用数据线少（并口8条 -> 串口1条）
    • 传输距离远
    • 速度传输慢（比并口还慢）

      2.1.3主要配件的识别与安装

主机箱

主机箱是电脑配件中的一部分

主机箱的结构

  • 外壳
  • 支架
  • 面板上的各种开关
  • 指示灯

主机箱的作用

  • 放置和固定各种电脑配件
  • 起一个承托和保护作用
  • 屏蔽电磁辐射

主机箱款式

  • AT

    • 全称BaBy AT

  • ATX

    • 是现在最常见的机箱
    • 空间大
    • 安装槽多
    • 扩展性好
    • 通风条件也不错

  • Micro ATX

    • 是在ATX机箱的基础上改进的
    • 体积比ATX小，进一步节省空间

  • BTX

    • 其架构支持窄版设计
    • 系统结构更紧凑
    • 更易于散热
    • 安装更加便捷

拆装机的注意事项

防静电

  • 特点

    • 能量小
    • 电压高

  • 危害

    • 严重影响无线电设备的正常运作
    • 容易导致接触不良等故障
    • 静电火花会点燃某些易燃物而发生爆炸

  • 对计算机的影响

    • 带阴极射线管的显示设备，当受到静电干扰时会引起图像紊乱、模糊不清
    • 会造成Modem、网卡、Fax等工作失常，打印机总线不顺等故障

轻拿轻放

防止液体进入配件

不可暴力拆装

机箱电源

是整个主机的能源提供者

将220V转化为计算机所需的5V和3.3V的直流电

最高输出电压为12V

主机电源的质量的好坏会对计算机运行产生非常重要的影响

机箱电源通常会为不同的设备提供各种接口

  • 主板接口
  • CPU电源接口
  • 软驱电源接口
  • IDE设备电源D形接口
  • SATA设备电源扁平接口

拆卸电源时要将电源连接线分别从主板、硬盘、光驱等设备上拔下，并整理好连接线，再卸下机箱背后固定的电源螺丝，最后从机箱内取出

安装机箱电源的顺序与拆卸的顺序相反

主板

主板由多个部件组成，这些部件哦谈过一块大的电路板连接在一起的，这块大板被称为主板

又叫

  • 主机板
  • 系统板
  • 母版

是微型计算机中最基本的，也是极其重要的部件之一

CPU、内存、各种接口卡，和外部设备都要靠主板来协调工作

如果主板性能不好，与其相连接的部件的性能就不能得到充分的发挥

在主板上有两块主控芯片

  • 南桥芯片

    • 一般位于PIC插槽上面

    • 负责I/O设备的挂接

      • PCI总线
      • USB
      • LAN
      • ATA
      • SATA
      • 音频控制器
      • 键盘控制器
      • 实时时钟控制器
      • 高级电源管理

    • 发展的方向是集成更多功能

      • 网卡
      • READ
      • IEEE 1394
      • 无线网络

  • 北桥芯片

    • 靠近CPU旁边，被散热片盖住的是北桥芯片
    • 负责与CPU联系
    • 控制内存
    • AGP
    • PCI数据在北桥内部传输
    • 提供对CPU的类型和主频
    • 系统的前端总线频率
    • 内存的类型和最大容量
    • ISA/PCI/AGP插槽、ECC纠错等支持
    • 整合芯片组的北桥芯片还集成了显示核心
    • 由于北桥芯片的发热量较大，因此都装有扇热芯片
    • 很多主板型号都是以北桥芯片命名的

主板用6-9颗螺丝固定在机箱上，也有不封主板使用塑料扣子或铜螺帽固定，安装时将主板平整的放入机箱内，注意主板上的定位孔与机箱底部的金属螺丝或塑料钉相对应

CPU

即中央处理器

组成部分

  • 运算器
  • 控制器
  • 寄存器

是计算机运算与控制中心

衡量CPU性能的主要技术的指标有

  • 字长
  • 主频
  • 缓存Cache

生产CPU的公司主要有AMD Intel 威盛 和中国龙芯

CPU的安装与拆卸

  • 安装CPU

    • 安装CPU前先观察CPU 的铭牌、针脚和主板上的CPU插槽
    • 将主板放在工作台上向外拎起弯钩样的主板CPU卡鞘
    • 再将CPU卡架拎起到底
    • 找到CPU缺角标记对应的主板三角形缺口
    • 将CPU缺角标记对应主板的三角形缺口位置
    • 合上CPU卡架，扳下CPU卡销
    • CPU的一角有一个缺口，安装时位置一定要对准
    • 安装好CPU后还要安装CPU风扇
    • 在CPU与风扇的接触面涂上硅脂，保证CPU与风扇的无缝接触，提高扇热效率

  • 先拆卸CPU风扇

  • 拔下CPU风扇的电源线

  • 拆卸主板底部的风扇底座

  • 拆卸CPU时先拉起卡销及卡架，轻轻拉到底，然后用手指捏住CPU边缘取出

内存

又称主存

是计算机的重要存储设备

在计算机通电运行时用于临时存储运行的程序和数据

特点

  • 传输速度快
  • 容量小
  • 价格高
  • 断电后内存中的数据将丢失

组成

  • 内存芯片
  • 电路板
  • 金手指

分类

  • SDRAM（同步动态随机存储器）

    • 已经被市场淘汰

  • DDR SDRAM（双倍速率同步动态随机存储器）

    • 占据了市场的主流

    • 发展出了

      • DDR2
      • DDR3
      • DDR4

  • RDRAM（采用芯片对芯片接口技术的存储器）

内存的 速度取决于内存的主频，即内存所能达到的最高工作频率

安装内存条的方法

  • 观察主板上的内存插槽
  • 用双手大拇指将内存插槽两边的卡栓向外扳到底
  • 注意将内存的方误插标记
  • 将内存的方误差标记与主板内存插槽的槽口对齐后，两手大拇指将内存插槽两边的卡栓用力往下压
  • 随着内存的插入，当听到咔的一声响，插槽两边卡栓自动弹起将内存条扣紧，内存安装完成

拆卸内存条

  • 双手大拇指同时向外扳开内存插槽两侧白色的卡栓，内存条会自动弹出

硬盘

硬盘的组成

  • 硬盘驱动器

    • 又称硬盘机或硬盘子系统
    • 是独立于主机之外的一个完整装置

  • 硬盘控制器

    • 俗称硬盘控制卡
    • 做成电路板的形式插在主机箱内主板的总线插槽中
    • 作用是接收主机发送的命令和数据，并转换成控制命令和驱动器可以接收的数据格式，以控制驱动器的读写操作

  • 盘片

    • 是信息存储的介质

    • 材质

      • 铝合金
      • 陶瓷
      • 玻璃
      • 基片上涂磁性材料

  • 把多片盘片固定在一根主轴上，称为盘组

  • 把磁头、电机、盘片等驱动部件甚至读写电路制成一个不可随意拆卸的整体，叫“头盘组合体”

  • 硬盘是 个高度密封的组合体，不可随意拆开

硬盘接口

  • 接口类型

    • IDE

      • 把主机和硬盘之间的接口电路与硬盘控制器整合在一起

      • 优点

        • 简化了主机方面连接硬盘的接口电路

      • 是一个接口标准，定义了主机与硬盘连接的40条信号线

      • 是设备级接口

      • 特点

        • 数据最大传输速率3MB/s
        • 一条IDE信号线最多连接两台IDE设备
        • IDE接口与ISA总线相连

      • IDE改进后的接口称为EIDE接口

        • 一般提供两个接口
        • 每个接口可以连接主从两个设备
        • 两个接口可以同时连接四个设备
        • 不仅限于硬盘，还支持CD-ROM等IDE设备
        • 支持的最大硬盘容量为8.4GB
        • 最大传输速率为16.7MB/s
        • 普遍使用PIO（程序输入输出）模式传送数据
        • 在PIO模式下，CPU可以直接进行读写控制

    • SATA

      • 即串行ATA接口

      • 作为一种新形的硬盘接口技术于2000年由Intel公司率先提出

      • 技术

        • 低电压差分信号技术

          • 数据传输率更大

          • 传送距离更远

          • 信号电压从并行ATA的5V降低到了SATA的0.7V

            • 降低了磁盘驱动器的功耗
            • 缩小了开关控制器的尺寸

        • 采用点对点拓扑结构

          • 可以为每个设备提供全部带宽
          • 提高了总体性能

        • SATA标准

          • SATA 1.0

            • 150MB/s

          • SATA 2.0

            • 300MB/s

          • SATA 3.0

            • 600MB/s

    • SCSI

      • 是一种智能化、通用型、系统级的接口

      • 适合并发数据处理

      • 可以把外部设备与一台或几台主机相连

      • 有5个通道，可以在6M或更长的距离内以菊花链的方式挂接7个不同的外设

      • SCSI-1

        • 最大外部传输率为5MB

      • SCSI-2

      • Ultra SCSI（也称SCSI-3）

        • 根据使用的频率和总线插槽分为

          • 快速

          • 宽带

          • 快速宽带

            • 80MB/s

      • 主要用在服务器上

硬盘的主要技术参数

  • 容量

    • 以兆字节（MB）、吉字节（GB）、太字节（TB）为单位

  • 传输率

    • 指硬盘的读写数据的速度

    • 单位是MB/s

    • 硬盘的数据传输率也称为持续传输率

      • 内部传输率
      • 外部传输率

  • 主轴转速

    • 内部传输率

      • 也称持续传输率
      • 反映了硬盘缓冲区未用时的性能
      • 取决于硬盘的转速

    • 外部传输率

      • 也称突发传输或接口传输率
      • 标称的是系统总线与硬盘缓冲区之间的数据传输率
      • 主要与硬盘接口类型和硬盘缓存大小有关

    • 转速

      • 7200rpm
      • 4200rpm
      • 5400rpm
      • 10000rpm

固态硬盘

  • SSD，简称固盘

  • 用固态电子存储芯片阵列而制成的的硬盘

  • 在外系上也完全与普通硬盘一致

  • 优点

    • 读写速度快
    • 防震抗摔性强
    • 低功耗
    • 无噪音
    • 重量轻

  • 缺点

    • 容量小
    • 价格高
    • 使用寿命短

光驱

是计算机用来读写光盘的设备

主要类型

  • DVD-ROM光驱
  • CD-RW刻录机
  • DVD-RW刻录机

  • IDE接口

光盘

  • CD-ROM

    • 只读形光盘存储器
    • 数据传输率以150KB/s为1倍速用X来表示
    • 最大的读取速度是56X

  • DVD

显卡

全称显示接口卡

又称显示适配器

是主机与显示器之间的接口电路

作用是把主机要显示的字符、图形等信息进过电路的转换，用显示器可以接受的形式送到显示器显示

分类

  • 集成显卡

    • 将显示芯片、显存等机场在主板上
    • 稳定性好
    • 兼容性强

  • 独立显卡

    • 以板卡的形式插在专用插槽上

    • 按接口类型分类

      • ISA显卡
      • PCI显卡
      • AGP显卡
      • PCI-E显卡

    • 有单独的显存，一般不占用系统内存

    • 缺点

      • 发热量大
      • 系统功耗有所加大
      • 需要额外花费购买显卡的资金

    • 拆卸独立显卡

      • 用螺丝刀顺时针旋出螺丝
      • 用拇指向外掰开显卡卡栓
      • 左右两只手同时按住板卡两侧垂直向上用力将显卡拔出
      • 注意不能左右摇晃，避免弄坏插槽
      • 不要让显卡金手指与金属物体接触

    • 安装独立显卡

      • 观察拆下的显卡和主板的显卡插槽
      • 将卡栓向外扳开
      • 双手按住显卡上方垂直向下插入槽中

组成部分

  • 显示芯片

    • 又称图形处理器（GPU）

    • 是显卡的大脑

    • 在显卡中如同计算机中的CPU

    • 其性能很大程度上决定显卡的档次和性能

    • 是2D显示卡和3D显示卡的区别依据

      • 3D显示芯片是将是三维图像和特效处理功能集中在显示芯片内，即所谓的“硬件加速”功能

BIOS与CMOS

BIOS

  • 是指微机的基本输入输出系统
  • 是一组固化在计算机主板上的一个ROM芯片上的程序
  • 基本输入输出程序（BIOS）
  • 主要保存计算机系统最重要的基本输入输出程序，系统底层设置，开机上电自检程序和系统启动自举程序等
  • 主要功能是给计算机提供最底层、最直接的硬件控制和设置
  • 断电后BIOS设置不会丢失

CMOS

  • 金属互补氧化物半导体存储器
  • 是计算机主板上一块可读写的RAM芯片，用来保存当前计算机系统的硬件配置和相关参数的设定信息
  • CMOS芯片由系统通过一块后备电池供电无论开机还是关机，CMOS中的数据都不会丢失

CMOS设置

开启或重启计算机，当屏幕出现自检画面时，根据屏幕提示按Delete或Del,就可以进入BIOS设置界面，在BIOS设置界面中，我们能了解CMOS版本名称以及一些常用按键的作用

目前BIOS有三种类型，即Award BIOS、AMI BIOS 和 Phoenix BIOS

版本越高的BIOS所支持的硬件就越多

CMOS设置

  • 查看计算机信息

    • 在BIOS设置中选择Standard CMOS Features

      • 可以查看计算机中

  • 修改BIOS的日期和时间

    • 在BIOS设置中选择Standard CMOS Features

  • 设置计算机启动时除键盘故障外检测到任何故障终止

    • 在BIOS设置中选择Standard CMOS Features

  • 设置启动顺序

    • Advanced BIOS Features

      • 高级BIOS功能设置

  • 设置开机密码

    • Set Supervisor Password

  • 清除BIOS密码

  • 恢复BIOS设置

  • 载入最安全设定

  • 电池放电法

    • 用金属导体短接电池座中的正负极
    • 再将电池放入

    • 用镊子将跳线帽从1、2脚换到2、3脚上，再插入回1、2脚

UEFI

统一可扩展固件接口

用于操作系统自动从预启动的操作系统环境加载到一种操作系统上

旨在代替BIOS，从而使开机程序化繁为简，节省时间，提高开机后操作系统的启动速度

此标准由140多个公司共同创建，包括微软

UEFI减少了BIOS的自检过程，所以能缩短开机时间，给用户带来良好开机体验

在UEFI模式下安装的系统只能用UEFI引导

在Legacy模式下安装的系统也只能在Legacy下引导

Legacy模式支持磁盘分区为MBR结构

特点

  • 通过保护预启动或预引导进程，抵御bookit攻击，从而提高安全性
  • 缩短了启动时间和从休眠状态恢复的时间
  • 支持容量超过2.1TB的驱动器
  • 支持64位的现代固件设备驱动程序，系统在启动过程中可以使用他们来对超过172GB内存进行寻址
  • UEFI可以与BIOS结合使用
  • 支持超过2.1TB的驱动器

MBR和GPT

MBR

  • 主引导记录
  • 又叫主引导扇区
  • 是开机后访问的第一个扇区
  • 无法支持超过2GB的分区

GPT

  • 全局唯一标识分区表
  • 提供更加灵活的分区机制
  • 允许大于2TB的容量
  • win8以后才有支持UEFT的

      6. 3.4.6. 在主板上有两块主控芯片
