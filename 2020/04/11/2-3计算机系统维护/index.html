2.3.1计算机系统维护的方法

计算机系统除尘

计算机系统运行时产生的静电很容易吸附灰尘，如果灰尘不及时清除，就会越积越多，当覆盖在板卡上的灰尘厚度达到一定的范围时，就会对微机系统造成很大 的影响

  • 灰尘会影响电源、CPU、显卡等配件上风扇的正常运行，使其旋转时发出很大的噪声
  • 灰尘会进入CPU转接卡、显卡、内存的插槽、会导致各部件之间接触不良，影响计算机的工作
  • 计算机元件上的灰尘多了会影响配件的散热，当计算机元器件因散热不好而导致电路损坏，使计算机无法正常工作。

计算机软件系统中速度变慢的原因和改进方法

  • 计算机软件在使用一段时间后启动速度和运行速度会变的越来越慢，造成系统变慢的原因有很多种，下面从软件方面来分分析系统速度变慢的一些原因，并系统相应的改进方法以提供参考

    • 在开机时候加载了太多的开机启动程序

      • 我们可以在“开始”-“所有程序”-“启动”程序中将不需要开机时启动的程序删除掉，有些自启动软件在-“启动”项中是找不到的，但直接通过该软件的“设置”功能或一些工具软件

    • 桌面上使用了华丽的主题或背景

    • 桌面上的图标或文档太多

    • 安装字体过多

    • 硬盘碎片过多

    • 删除程序不彻底

计算机硬件故障诊断的方法

观察法

  • 观察是维修判断汇总某部分的工作异常造成的，它贯穿于整个维修过程

软件诊断法

  • 当计算机出现运行不稳定的情况时，可以用专门的检测软件来对电脑软，硬件进行检测，如3D Mark 、鲁大师等，通过这些软件反复测试而生成的报告文件，就可以比较轻松的找到一些由于运行不稳定而引起的计算机故障，此外，通过检查操作系统的部分信息，也能诊断和排除一些故障

插拔法

  • 是通过将怀疑有故障的板卡或接头拆卸后，将接触部分（如金手指）清洁干净后再重新安装来寻找故障位置的方法。用于解决由于板卡或插头接触不良引起的计算机黑屏、死机等故障，板卡后故障解决，一般可以断定问题就处在这个设备或相关接口上
  • 插拔法要反复开关机，如不涉及硬盘故障，可以先将硬盘电源线拔掉，避免由于短时间内频繁的开关机对硬盘造成的损伤

最小系统法

  • 最小系统法是指从维修角度判断能使计算机开机运行的最基本的硬件和软件环境，主要用来判断在最基本的软硬件环境中，系统能否正常工作

    • 启动型：电源+主板+CPU+内存
    • 点亮型：电源+主板+CPU+内存+显卡+显示器
    • 进入系统型：电源+主板+CPU+内存+显卡+显示器+硬盘+键盘

替换法

  • 用好的部件去代替怀疑有问题的部件，以判断故障现象是否消失的一种维修方法。也可以用替换法将怀疑有故障的部件换到正常的机器中测试，从而有效的判断出故障点

诊断卡法

  • 诊断卡法是利用专用的诊断卡对系统进行检查的方法。
  • 将诊断卡插在主板插槽中，然后开机，诊断卡上的指示灯会显示出电脑各个部件的工作情况，根据诊断卡上指示灯的显示情况，对照诊断卡的说明书，可以轻松鉴别出有故障的部件

敲打法

  • 一般用在怀疑某设备有接触不良的故障时，通过振动、适当的扭曲，甚至敲打的方法使故障复现或消失，从而判断故障的位置

计算机系统维护概念

计算机系统维护是指为保证计算机系统能够正常运行而进行的定期检测、修理和优化

他包括计算机硬件维护和计算机软件维护两个方面

  • 硬件维护主要是指对计算机主要部件的除尘保养
  • 计算机软件维护包括计算机操作系统的优化和病毒等的查杀

2.3.2注册表的概念和功能

注册表的概念

注册表（Regedit）是Windows中的一个重要的数据库，他储存了Windows系统和应用程序的各种配置信息，用户可以通过修改注册表中的配置信息实现改变系统设置的目的

注册表最早出现在Windows3.0推出的OLE技术的时候，但从Windows95开始，注册表才真正的称为了Windows用户经常接触的内容，并在其后的操作系统中继续沿用至今

注册表的作用

注册表中存放着软硬件的各种参数，直接控制着Windows的启动、硬件驱动程序的装卸和一些Windows应用程序的运行，从而在整个系统中起着核心的作用。

它包括

  • 软硬件的有关配置和状态信息，应用程序和资源管理器外壳的初始条件、首选项和卸载数据等；
  • 联网计算机整个系统的设置和各种许可
  • 文件扩展名与应用程序的关联
  • 硬件部件的描述、状态和属性、性能记录；
  • 其他底层的系统状态信息，以及其他数据等

如果注册表受到破坏，轻则系统的启动出现异常，重则整个系统完全瘫痪

认识注册表的组成

注册表由注册数据库和注册表编辑器两个部分组成。

注册表数据库在Windows 7系统中分为用户配置文件和系统配置文件两个部分。

  • 用户配置文件包括两个隐藏系统文件Ntuser.dat和Ntuser.ini以及日志文件Ntuser.log等，保存在根文件夹“用户”下，每个用户下相应都有自己的用户配置文件；
  • 系统配置文件包括Default，Software，System，APPevent.evt，SecEvent.evt及SysEvent.evt等多个隐藏系统文件及相应的“*.log”文件等，系统配置文件存放在Windows系统目录下的“System32\Config”中

注册表文件可以通过注册表编辑器工具（Regedit.ext）进行查看和维护，注册表编辑器与资源管理器的界面相似

Windows 7 操作系统中用注册表编辑器打开的注册表窗口，左边窗格中，可以看到所有数据以一种树状结构展开，由“计算机”开始，分为五个分支，每个分支名都以HKEY开头，称为根键（KEY）每个跟键分别包含的信息如下

  • HKEY_CLASSES_ROOT（种类_根键）：包含了启动应用所需的全部信息，主要包括软件的配置信息，特别是对象链接与嵌入（OLE）和Windows外壳应用程序使用的信息等
  • HKEY_CURRENT_USER（当前_用户键）：用于记录当前登录到计算机上用户的相关信息
  • HKEY_LOCAL_MACHINE（本地_机器键）：用于记录当前登录到计算机上用户的相关信息
  • HKEY_USERS（用户键）：用于保存缺省用户（.DEFAULT）的配置文件于当前登录用户的配置设置。
  • HKEY_CURRENT_CONFIG（当前_配置键）：该键包含计算机系统硬件的当前配置情况，实际上也是指向HKEY_LOCAL_MACHING/Config结构中的某个分支指针，其下面的子关键字及内容与HKEY_LOCAL_MACHING/Config/0001分支下面的子键内容是完全相同的
  • 每个根键内以键和子键的方式组织起对各类数据的访问，类似于资源管理器中的文件与文件夹。键展开后可以看到下面还包含下一级键或子键，就像文件夹下面还有子文件夹一样。当单击某一键或子键时，右边窗格中显示的是所选键内包含一个或多个键值项，每个键值项类似于文件夹的文件，它由键值名、键值类型和键值三部分组成。根据键值项的不同，键值项分为字符型和数值型两大类

打开注册表编辑器

最简单的打开方法：在“开始”中——“搜索”中输入“regedit”就可以了

另外一种方法：在Windows目录下找到regedit.exe程序文件，直接双击打开

操作方法：选择“文件”菜单的“导出”命令，然后在弹出惨淡选择导出文件的位置，输入导出文件的文件名

恢复注册表

操作方法1：双击备份注册表

2.3.3Ghost软件应用

Ghost

是美国赛门铁克公司推出的一款出色的硬盘备份还原工具，可以实现FAT16，FAT32，NTFS，OS2等多种硬盘分区格式的分区及硬盘的备份还原，俗称克隆软件。在计算机软件的维护汇总常用备份和还原

操作系统备份

使用Ghost软件可以将操作系统所在分区以文件的形式备份到硬盘，备份操作系统的具体设置为

  • 用自启动光盘引导系统，进入CMOS设置界面，将第一引导顺序设置为CDROM，第二引导顺序设置为硬盘，再将带有Ghost软件的自启动光盘放入光驱，按“F10”保存CMOS设置后退出BIOS；计算机重新启动，被光驱成功引导后会弹出一个窗口，启动Ghost程序
  • 执行Ghost菜单中备份分区到镜像文件（.GHO）的菜单命令。
  • 在弹出的“选择备份分区”对话框中依次选择需要备份的硬盘和硬盘分区。
  • 在弹出的备份路径选择对话框中依次选择备份文件保存位置和备份的镜像文件名
  • 在弹出的对话框中选择备份文件是否进行压缩。“NO”表示亚这样备份的文件容量比其他两种方法要大；“FAST”表示压缩比例小但执行备份的速度快；“High”表示压缩比例较高但执行备份的速度慢
  • 备份文件的过程等进度条达到100%后完成分区的备份
  • 备份文件结束后，退出Ghost,重启计算机

操作系统还原

使用Ghost软件可以将备份的Ghost镜像文件还原到C盘，还原操作系统的具体方法如下

  • 用自启动光盘引导操作系统

    1. 2.1. 注册表的概念
      1. 2.1.1. 注册表（Regedit）是Windows中的一个重要的数据库，他储存了Windows系统和应用程序的各种配置信息，用户可以通过修改注册表中的配置信息实现改变系统设置的目的
      2. 2.1.2. 注册表最早出现在Windows3.0推出的OLE技术的时候，但从Windows95开始，注册表才真正的称为了Windows用户经常接触的内容，并在其后的操作系统中继续沿用至今
    2. 2.2. 注册表的作用
      1. 2.2.1. 注册表中存放着软硬件的各种参数，直接控制着Windows的启动、硬件驱动程序的装卸和一些Windows应用程序的运行，从而在整个系统中起着核心的作用。
      2. 2.2.2. 它包括
      3. 2.2.3. 如果注册表受到破坏，轻则系统的启动出现异常，重则整个系统完全瘫痪
    3. 2.3. 认识注册表的组成
      1. 2.3.1. 注册表由注册数据库和注册表编辑器两个部分组成。
      2. 2.3.2. 注册表数据库在Windows 7系统中分为用户配置文件和系统配置文件两个部分。
      3. 2.3.3. 注册表文件可以通过注册表编辑器工具（Regedit.ext）进行查看和维护，注册表编辑器与资源管理器的界面相似
      4. 2.3.4. Windows 7 操作系统中用注册表编辑器打开的注册表窗口，左边窗格中，可以看到所有数据以一种树状结构展开，由“计算机”开始，分为五个分支，每个分支名都以HKEY开头，称为根键（KEY）每个跟键分别包含的信息如下
